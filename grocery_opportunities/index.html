Grocery Opportunities

Python pipeline that runs weekly (via GitHub Actions) and:

  • Scrapes grocery pricing data
  • Pulls Google Trends for food keywords
  • Generates derived opportunity ideas and scores
  • Pushes structured data into Notion via API
  • Flags opportunities where score > threshold and sets Status to Decision Required

Results are written to a Notion database each run. Configure NOTION_API_KEY and NOTION_DATABASE_ID in the repo’s GitHub Actions secrets to enable the Notion push.
